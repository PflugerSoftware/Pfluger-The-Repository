**PART 1**

# Interactive Mapping Exercise

+-----------------------------------------------------------------+
| **Overview**                                                    |
|                                                                 |
| This in-person activity uses a large, printed Google Map of the |
| Baytown campus mounted on a board. Participants receive colored |
| dot stickers and place them on the map in response to guided    |
| prompts. Each dot color corresponds to one of Lee College's     |
| five guiding principles, plus a sixth color for barriers and    |
| challenges.                                                     |
+=================================================================+

## A. Dot Color System --- Aligned to Guiding Principles

Each dot color is tied directly to a guiding principle. Distribute this
legend to all participants before beginning.

  ----------------------------------------------------------------------------
    **DOT**    **GUIDING           **USE THIS COLOR WHEN\...**
               PRINCIPLE**         
  ------------ ------------------- -------------------------------------------
    **BLUE**   **People-Centered   Mark places where you feel a sense of
               Campus**            belonging, inclusion, safety, or support

   **GREEN**   **Future            Mark spaces with untapped potential,
               Evolution**         flexibility, or opportunities for
                                   redevelopment

   **ORANGE**  **Build for         Mark areas that support or could better
               Opportunity**       support workforce training, academics, or
                                   partnerships

   **PURPLE**  **Community         Mark spaces that bring people together or
               Engagement**        strengthen campus-community connections

    **GOLD**   **Enduring          Mark places that embody Lee College\'s
               Identity**          character, history, and lasting
                                   institutional spirit

    **RED**    **Barriers /        Mark locations with physical barriers,
               Challenges**        safety concerns, or obstacles to teaching
                                   and learning
  ----------------------------------------------------------------------------

## B. Participant Instructions

+-----------------------------------------------------------------+
| **Instructions for Participants**                               |
|                                                                 |
| 1.  You will receive 6 colored dot stickers --- one for each    |
|     color in the legend above.                                  |
|                                                                 |
| 2.  A facilitator will read each prompt aloud. Listen to the    |
|     prompt, then place the dot of the matching color on the     |
|     campus map location that best responds to the prompt.       |
|                                                                 |
| 3.  Place each dot as precisely as possible --- on a specific   |
|     building, outdoor space, pathway, or parking area.          |
|                                                                 |
| 4.  If you feel strongly about a location, you may place more   |
|     than one dot of the same color, but try to be selective.    |
|                                                                 |
| 5.  There are no right or wrong answers. Your lived experience  |
|     on this campus is exactly what we want to capture.          |
|                                                                 |
| 6.  This exercise takes approximately 5--7 minutes.             |

# Survey Summary & Design Rationale

**Key Insights This Survey Is Designed to Uncover**

- Where faculty perceive the strongest and weakest areas of safety,
  inclusion, and belonging across specific campus zones

- Which classroom and lab environments are limiting instructional
  effectiveness --- and what types of improvements would have the
  greatest impact

- How well the current campus layout and facilities support workforce
  training, transfer pathways, and academic cohesion

- Where collaboration between faculty, departments, and the broader
  community is occurring --- and where it is being constrained by the
  physical environment

- How faculty perceive Lee College's physical identity and what
  investments in design and architecture would most strengthen
  institutional pride

- Which campus areas have the greatest potential for transformation and
  long-term adaptability
